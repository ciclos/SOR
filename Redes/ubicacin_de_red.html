Ubicación de red

La ubicación de red en Windows permite aplicar configuraciones de seguridad a
las conexiones de forma rápida, simplemente identificando la conexión como
domestica, trabajo o publica.
La primera implicación de seguridad es que se modifican las reglas del firewall
siendo las mas restrictivas para la ubicación publica y las menos para la domestica.
