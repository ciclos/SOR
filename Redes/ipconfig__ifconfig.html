ipconfig / ifconfig

ipconfig

El comando ipconfig en Ms Windows nos permite obtener información de la configuración de la interfaz de red.

Con el parametro /all obtenemos una versión mas completa de la información.

Entre esa información podemos encontrar:

  • Dirección IP del host para cada interfaz.
  • Mascara de red
  • Puerta de enlace predeterminada
  • Dirección hardware de cada interfaz.

Una opción interesante de este comando (en Windows ) para la administración de redes es la opción /displaydns y /flushdns, que nos permite ver la cache DNS y borrarla respectivamente.

¿Como lo hago?

ifconfig

En linux el comando es ifconfig y funciona básicamente igual que ipconfig con pequeñas diferencias. La primera que te saltará a la vista es que, por defecto, el comando no termina y manda continuamente peticiones de eco. Para pararlo presiona Ctrl+c.
