Crea 4 usuarios en linux y para cada uno de ellos crea un diretorio compartido en el servidor.

En un banco se nos ha pedido crear un sistema de tal forma que los administrativos puedan crear usuarios en el servidor simplemente escribiendo los nombres en un archivo de texto, llamado usuarios.txt.

Hemos pensado en crear un script que haga lo siguiente:

  • El archivo usuarios.txt que se va a leer, se ubicará en una carpeta compartida en el servidor al que los equipos clientes tendrán acceso.
  • El script se ejecutará cada cinco minutos usando contrab
  • El script debe comprobar primero si el usuario ya existe, si no existe lo crea.
  • Se compartira el home de los usuarios para hacerlo accesible mediante NFS.
  • Para no volver a crear los directorios una y otra vez debemos borrar el contenido del archivo de nombres.txt cada vez que lo usemos
  • El script crear un log donde guarda la hora de creación de cada usuario.

Crea una red interna en virtualbox para virtualizar la situación. Cuando tengas montanda la red, inicia el servidor sin pantalla y conectate por medio de ssh al mismo. Haz todas las gestiones en el servidor mediante ssh.

Finalemente crea un directorio compartido para todo el mundo y accede a él desde Windows 7.

Crea un script llamado crearUsuario.sh, que reciba por parametros el login de un usuario, su password y su directorio de home.
Si el usuario no existe lo crea y muestra por pantalla: "El usuario talYtal se creó correctamente". Si el usuario ya existe da el mensaje de error: "El usuario talYtal ya existe, no se ha creado". Recuerda darle permisos a ese usuario sobre sus directorio.

Crea un script llamado compartir.sh que reciba como parametros el nombre de un directorio. El script comprueba si el directorio ya está compartido, en cuyo caso da el siguiente mensaje: "El directorio talYtal ya estaba compartido". Si el directorio no estaba compartido, lo camparte y muestra el mensaje: "Directorio talYtal compartido correctamente"

Crea un script llamado pedirDirectorios.sh que pida al usuario la ruta de 5 directorios y meta esas rutas en un archivo llamado directorios.dat. Una ruta por linea.

En una empresa de software quieren tener un sistema de backup de tal forma que los "home" de los empleados se compriman en .tar, todos los dias a las 6:00 y se copien a un directorio compartido en el servidor. Implementa la solución usando NFS.

Se ha establecido la siguiente organización en una empresa: el jefe crea un archivo con las tareas de cada empleado, este archivo consiste en una lista de nombres de archivo que el empleado debe rellear. Implementa un sistema de tal forma que cuando el empleado inicie sesion, se ejecute un script que lea el archivo que hay en el servidor para ese usuario concreto y cree en el directorio /home/tareas todos los archivos indicados en la lista que contiene el archivo del servidor. Antes de crear los archivos hay que borrar lo que tuviera ese directorio. Implementa la solución usando NFS.

Crea un script que haga lo siguiente:

  • Imprima una cabecera y un menu de las cosas que puede hacer el script. Las opciones son las siguientes:
  • 1) Crear un usuario, con un directorio compartido a partir del nombre de usuario.
  • 2) Crear una cantidad de usuarios especificada por el usuario, a partir de un nombre de usuario. Se crea un directorio compartido para cada usuario.
  • 3) Borrar un usuario y su directorios compartidos
  • 4) Salir
  • El script se ejecuta constantemente hasta que el usuario elija la opción de salida
  • El script crea un archivo de log donde registra todo lo que va haciendo.
