Programación en Batch

Archivos .bat

En Microsoft Windows un archivo batch es un archivo de procesamiento por lotes. Se trata de archivos de texto sin formato, guardados con la extensión BAT que contienen un conjunto de comandos MS-DOS.
Este archivo puede ejecutarse desde la línea de comandos o haciendo doble click en él.

Estructura básica

Uno de los script más basico que puede escribirse es el siguiente: @echo off
echo Hola Mundo
pause
exit
 En este script podemos encontrar varios comandos utiles de control del mismo, como son:
  • @echo off. Con esta instrucción no se imprime por pantalla los comandos que esta ejecutando el script
  • pause. Detiene la ejecución a la espera de que se presione una tecla.
  • exit. Termina la ejecución

Variables

Las variables se asignan por medio de la palabra reservada set. Por ejemplo, el siguiente script asigna una variable y despues imprime por pantalla su contenido. @echo off
set Nombre= Harvin Tellez
echo %Nombre%
pause

Obtener valores por teclado

Si queremos que el usuario nos de información por teclado usaremos una construcción parecida a la anterior, como se ve en el siguiente ejemplo: @echo off
set /p Nombre=Escriba su Nombre:
echo %Nombre%
pause En este script se mostrará el mensaje: Escriba su Nombre. El script se quedará esperando la entrada por teclado.

Instrucción GOTO y condicional

La instrucción goto nos permite saltar de una línea a otra dentro del codigo. Para ello primero necesitamos etiquetar puntos de aterrizaje de la instrucción. Esto se hace anteponiendo dos puntos a un nombre ( el nombre del punto de aterrizaje). Después simplemente se nombra ese punto de aterrizaje después de la instrucción goto.
Esta instrucción se suele usar en conjunción con el condicional que permite comprobar una condición, de tal forma que si se cumple saltamos y en caso contrario no. Las comparaciones mas comunes son las siguientes:
  • if variable==valor (para expresar si la variable es igual a valor haz esto)
  • if variable LSS valor (para expresar si la variable es menor a valor haz esto)
  • if variable GTR valor (para expresar si la variable es mayor a valor haz esto)
  • if variable LEQ valor (para expresar si la variable es menor o igual haz esto)
  • if variable GEQ valor (para expresar si la variable es mayor o igual haz esto)

A continuación podemos ver un ejemplo sencillo de uso de estas dos instrucciones. @echo off
set /p variable=¿Cuantos son 2 +2 ?
cls
if %variable%==4 goto correcto else goto incorrecto
:incorrecto echo te has equivocado
goto terminar :correcto
echo correcto!
:terminar pause
exit
