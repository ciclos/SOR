Redirección

Redireccionar significa capturar la salida de un archivo, comando, programa o script y dárselo como entrada a otro comando, programa o script.

Las redirecciones mas comunes se indican de la siguiente forma:

  • comando > archivo Redirecciona la salida del comando a un archivo y destruye lo que tuviera el archivo.
  • comando >> archivo Redirecciona la salida del comando a un archivo y le añade la salida a lo que tuviera el archivo.

Actividad

  1. Crea un script que liste los archivos y directorios del directorio actual y lo guarde en un archivo.
  2. Crea un script que liste los archivos y directorios del directorio actual y lo guarde en un archivo. Cada vez que ejecutemos el script se añade al archivo.
  3. Crea un script que coja como parametros un usuario y una contraseña y añada estos datos a un archivo llamado usuarios.txt.
  4. Crea un script que liste el contenido de /etc/network /bin y /home y cree un archivo con toda esa información.
  5. Crea un script que obtenga por parametros una dirección IP y que le haga un ping de un solo eco y que guarde el resultado en un archivo.
