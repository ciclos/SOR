Parametros

Los parámetros posicionales o argumentos son valores que acompañan a la invocación del comando y que permiten darle datos de entrada al comando para que trabaje con ellos. Por ejemplo, podemos usar el comando para listar el contenido de un directorio sin parametros:

Nuestros script pueden tener parametros. Podemos acceder a ellos mediante sus nombres, que son: $1, $2, $3, ... Como ves lo que se indica es la posición del parametro, pues los parametros se ponen detras del nombre del script, como una lista separada por espacios. Actividad

  1. Crea un script que coja por parametros una frase y la muestre desordenada.
  2. Modifica el anterior para que primero se muestre la frase introducida y en la siguiente línea desordenada.
  3. Vuelve a modificar el anterior para que además antes de cada frase salga una indicación diciendo cual es cada una. Por ejemplo: 
    Frase ordenada: Hola, esta la ordenada.
    Frase desordenada: esta Hola, ordenada la
  4. Crea un script que muestre por pantalla los parametros introducidos, indicando delante de cada uno cual es su posición.
