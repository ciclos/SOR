Variables

Una variable es una posición de memoria donde podemos almacenar información.

En los scripts de shell Bash una variable se define mediante un nombre. Los nombres de variables no puede contener espacios, por lo que si consiste en varias palabras se usa el estilo de escritura Camel Case. Según este estilo en un nombre compuesto la primera de cada palabra se escribe en mayúsculas, excepto la primera palabra.

Para asignar un valor a una variable se usa el simbolo =. El valor de la variable puede cambiar tantas veces como se defina en el programa. Para acceder al valor de una variable se usa el símbolo de dolar.

También podemos asignar el valor del resultado de un comando, introduciendo el comando entre estos simbolos $( ).

Interacción con el usuario

La utilidad de las variables es que varien su valor según las necesidades del usuario. Una de posibilidad para que esto suceda es que le pidamos al usuario que introduzca de forma interactiva los valores de las variables. Esto se consigue con la instrucción read.

Actividad

  1. Crea un script que almacene en variables los siguientes datos del registro de cada uno de los huespedes de un hotel:
    1. Numero de habitación
    2. Tipo de habitación ( cantidad de huespedes )
    3. Precio de la habitación
    4. Nombre
    5. PrimerApellido
    6. SegundoApellido
      Al ejecutar el script se debe mostrar por pantalla estos datos, con la indicación de que son cada uno.
  2. Modifica el script anterior para introducir el valor de las variables por parametros y que los datos se escriban en un archivo, que irá creciendo con cada ejecución del script.
  3. Modifica el script anterior para introducir el valor de las variables utilizando read.
  4. Crea un script que pida al usuario introducir una IP y haga un ping de 3 ecos, mostrando la respuesta por pantalla.
  5. Crea un script que pida 5 numeros por teclado y al final muestre la suma todos, la resta de todos a 100 y la multiplicación de todos.
  6. Crea un script que haga lo siguiente:
    1. Coge por parametros un nombre y crea un directorio con ese nombre en el home del usuario. (Podemos acceder directamente al home con el simbolo ~ )
    2. Pide por teclado 5 correos electronicos y sus contraseñas. Al final almacena todo en un archivo de texto.
    3. Comprime el archivo anterior y lo copia a la carpeta que se creó en el punto 1.
      Ejecuta el script 3 veces con diferentes datos para comprobarlo.
