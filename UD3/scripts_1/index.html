Scripts en Bash Shell

Un script es un archivo de texto que contiene una serie de comandos interpretables por el sistema en el que se va a ejecutar. Es importante que este archivo sea un archivo de texto sin formato, pues lo que se ejecutan son ordenes como si se usara la linea de comandos manualmente, de tal forma que cualquier carácter no imprimible distinto del espacio o retorno de carro hará el archivo no interpretable.

Estos archivos contienen un comando (o agrupación de comandos por tuberias) por línea.

El archivo se va ejecutando linea a linea.
