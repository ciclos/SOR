tail

Este comando presenta por defecto las ultimas 10 lineas de un archivo. Suele ser muy útil para consultar archivos de configuración o de logs del sistema. 

Por ejemplo, supongamos que acabamos de añadir un usuario al sistema y queremos comprobar si ha salido bien, podemos usar el comando de la siguiente forma: tail /etc/passwd. O podemos consultar las ultimas entradas del log del sistema: tail /var/log/syslog.
