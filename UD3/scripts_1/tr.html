tr

Este comando permite cambiar unos caracteres por otros. Aunque normalmente una de sus funciones mas usuales en la administración de sistemas es comprimir los espacios, es decir, quitar los espacios entre palabras para dejar solo uno.

Por ejemplo, el comando : echo "Hola     como     estas?" | tr -s ' ' , daría como resultado: Hola como estas?

Esto es muy útil cuando se usan otras herramientas, como cut, donde se debe indicar un delimitador, pues no es lo mismo un espacio que varios. Por ejemplo,  echo "Hola     como     estas?" | cut -f 2 -d " ", no nos daría el resultado esperado, a no ser que hagamos: echo "Hola     como     estas?" | tr -s ' ' |cut -f 2 -d " "
