grep

El comando grep busca dentro de un archivo las líneas que contengan el patrón que se le indica por parámetros.

Ejemplos

  • grep "ejemplo" * listara las lineas que concuerden con la cadena ejemplo de "todos" los archivos del directorio actual.
  • grep ‐i hola pp.txt considera idénticas las mayúsculas y minúscula en la cadena de búsqueda hola. (hola=HOLA=Hola=etc.).
  • grep ‐ri "hola" ./ busca en los archivos del directorio actual y los subdirectorios la palabra hola.
  • grep ‐v hola pp.txt lista las líneas que no contengan la cadena hola del archivo pp.txt
  • grep ‐n hola pp.txt las líneas concordantes con la cadena hola del archivo pp.txt se mostrarán acompañadas del número de línea.
  • grep ‐w sobre pp.txt listara solamente aquellas líneas que contienen concordancias que forman palabras completas.
  • grep [123] pp.txt listara las lineas que contengan los números 1, 2 o 3
  • grep '^L' pp.txt lista las lineas de pp.txt que comiencen la línea con el carácter L, el carácter ^ denota inicio de cadena.
  • grep 'h$' pp.txt lista las líneas de pp.txt que terminen en h, el carácter $ denota fin de cadena.

Actividad

  1. Crea un script que muestre todos los archivos de un determinado usuario, listado desde el directorio en que se encuentra el script. Crea una estructura de directorios de prueba si es necesario.
  2. Crea un script que muestre todos los archivos de un determinado usuario, listado desde el directorio raiz.
  3. Crea un script que muestre todos los archivos de un determinado usuario, listado desde el directorio raiz y los permisos que tienen esos archivos. ( Consulta el manual de ls para saber como listarlo todo desde la raiz)
  4. Crea un script que muestre todos los archivos de un determinado usuario introducido por parametros, listado desde el directorio raiz y los permisos que tienen esos archivos.
  5. Crea un script que muestre todos los archivos de dos usuarios.
  6. Crea un script que muestre todos los archivos de varios usuarios introducidos por parametros, desde el directorio raiz y mostrando los permisos. Ten cuidado de introducir exactamente la cantidad de parametros que necesites, porque si falta alguno grep tomará el espacio como un patrón de busqueda.
  7. Crea un script que mande un ping de un solo eco a todas las IPs de la red y que guarde en un archivo una línea por cada equipo que responda al ping. En esa linea debe aparecer la IP del host que responde.
