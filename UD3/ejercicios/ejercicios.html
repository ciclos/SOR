El comando adduser es mas comodo de usar de forma interactiva, porque te pide el password y automaticamente crea un directorio en el /home. En cambio useradd, no hace esto por defecto, pero es mas util para incluirlo en un script. Useradd permite asignar a un usuario un directorio previamente creado, para que al hacer login se inicie en él. Por otro lado 'passwd' nos permite cambiar la clave de un usuario.
Crea un script que haga lo siguiente:
  • Crea un usuario utilizando el nombre introducido por parametros.
  • Crea un directorio en /home para ese usuario con el nombre del usuario.
  • Cambia el propietario y grupo del directorio para darle acceso al usuario.
  • Llama a passwd para ponerle una contraseña al usuario.
  • Crea un script que nos permita ver una lista de los nombres de todos los usuarios del sistema. Antes de mostrar esta lista, por pantalla se debe mostrar el mensaje: USUARIOS.
  • A continuación se muestran todos los grupos existentes en el sistema, mostrando solo el nombre. Antes de mostrar esta lista, por pantalla se debe mostrar el mensaje: GRUPOS.
  • Modifica el script y que los usuarios y grupos salgan en ristra, separados por comas.
  • Crea un archivo de registro del script, es decir, redirije la salida a un archivo llamado: user.log
