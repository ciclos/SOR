Archivos de configuración

Linux se configura por medio de archivos de texto. Algunos de los mas importantes son:

  • /etc/host: Contiene resoluciones DNS estáticas y únicas del usuario. Como mínimo contiene una entrada para localhost.
  • /etc/resolv.conf: Contiene las direcciones IP de los servidores DNS. Se actualiza automáticamente si se configura la interfaz como DHCP o puede que contenga 127.0.0.1.  En todo caso solo debería tocarse para configurar manualmente nuestro servidor DNS ( aunque eso también podemos hacerlo desde el archivo de interfaces )
  • /etc/network/interfaces: se utiliza para definir los nombres lógicos de las interfaces de redes, a los que se les asociará una configuración determinada. El comando ifup es el encargado de relacionar una interfaz física con una interfaz lógica.
