Monitorización Linux

La monitorización de Linux puede hacerse de muchas formas, pero la mas común es mediante la consulta de los archivos de configuración y log de Linux. ( en Linux todo son archivos de texto ). Si la consulta es puntual lo normal es hacer una conexión SSH y comprobar ciertos parámetros, pero lo mas normal es crear scripts que nos permitan automatizar las tareas periódicas.

Este documento vamos a ver algunos de los archivos de configuración y log de Linux mas usuales.
